## ✨ Overview

**Nailed by Faith** is a self-contained landing + booking site built as a single `index.html` file.
It pairs a pastel, glassmorphic aesthetic with a real backend: clients can register, browse the
gallery and pricing, and request appointments, while the studio owner manages everything from a
built-in admin portal.

## 🌸 Features

| | |
|---|---|
| 🖼️ **Auto-scrolling gallery** | Dual infinite-scroll tracks that pause on hover |
| 📅 **Booking calendar** | Date picker with request review and confirmation flow |
| 🔐 **Accounts** | Email/password + Google sign-in via Firebase Auth |
| 🛠️ **Admin portal** | Owner-only dashboard to review and manage bookings |
| 💖 **Glassmorphic UI** | Blurred nav, gradient text, soft pastel washes |
| 📱 **Fully responsive** | Fluid `clamp()` sizing scales from phone to desktop |
| ☎️ **Contact** | One-tap phone & email links in the footer |

## 💎 Services

- **Gel Sets** — glass-like finish, shaping & cuticle care *(from $15)*
- **Nail Art** — hand-painted custom designs *(from $5 add-on)*
- **Rhinestones & 3D** — applied per finger *($2 / finger)*
- **Blooming Gel** — ethereal ink-drop effect *($1 / finger)*

## 🛠️ Tech Stack

- **Vanilla JavaScript** — no frameworks, no bundler
- **Firebase** — Authentication + Firestore (v11.10.0, ES modules via CDN)
- **CSS** — custom properties, fluid typography, glassmorphism
- **Hosting** — GitHub Pages friendly (just drop in the file)

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/CameronCodesStuff/nailed-by-faith.git

# Open it — that's it
open index.html
```

> Configure your own Firebase project credentials inside the `<script type="module">`
> block, and set the admin email to grant portal access.
